Le Memory est un jeu très facile !

Memory en ligne, jeu de mémoire ou encore appelé jeu des paires
exerce ta mémoire en retrouvant les couples d'images identiques.
Pour cela, tu dois cliquer sur deux cartes successivement.
Si tu formes correctement les couples, les images restent apparentes.
Si tu te trompes, les cartes redeviennent faces cachées. Le jeu se termine lorsque toutes les cartes sont retournées.
